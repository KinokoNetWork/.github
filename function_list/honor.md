# Life鯖に追加予定の「二つ名機能」が完成しました。
`/honor list`で、自分が設定できる、二つ名を確認でき、<br>
`/honor set <特定のid>`で、設定されたPrefixをつけることができます。<br>
投票数、イベントやカスタム進捗等で、取り入れる予定です。
